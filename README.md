# 📌 README.md

Добро пожаловать в репозиторий с набором скриптов для расширения функционала PyRevit, ориентированного преимущественно на инженерные расчёты и проектирование систем отопления, вентиляции и кондиционирования воздуха (ОВиК).

## 🎯 Цель проекта

Данный репозиторий служит хранилищем для разработанных пользователями и сообществом скриптов, расширяющих стандартные возможности PyRevit. Основной целью является повышение эффективности работы инженеров-проектировщиков, занимающихся системами ОВиК, путём автоматизации рутинных операций, ускоряя процессы проектирования и повышая качество результатов.

## ⚙️ Особенности

- ✅ Автоматизация повторяющихся задач, таких как расчёт гидравлических сопротивлений, подбор оборудования и генерация спецификаций.
- 🛠 Удобные инструменты для визуализации проектов и автоматизированного формирования чертежей.
- 🔧 Возможность быстрого прототипирования новых инструментов и их интеграции в рабочий процесс.
- 💬 Поддержка сообщества разработчиков и пользователей, обмен опытом и лучшими практиками.

## 📦 Доступные скрипты

### 🏷️ MarksOn3D — Марки на 3D видах и планах

**Расположение:** `BIM_Rage_4er.extension/pyScript.tab/ОВиК.panel/MarksOn3D.pushbutton`

**Версия:** 2.0

**Назначение:** Автоматическая расстановка марок элементов ОВиК на 3D-видах и планах этажей.

**Возможности:**
- ✅ Поддержка 3D видов и планов этажей
- ✅ Раздельные настройки марок для 3D и планов
- ✅ Автоматическая фильтрация воздуховодов (по одному на группу система/сечение)
- ✅ Гибкие настройки размещения (ориентация, выноска, смещение)
- ✅ Сохранение настроек между запусками
- ✅ Подробное логирование процесса выполнения

**Поддерживаемые категории:**
- Воздуховоды
- Гибкие воздуховоды
- Изоляция воздуховодов
- Воздухораспределители
- Арматура воздуховодов
- Оборудование

📖 [Подробная документация](BIM_Rage_4er.extension/pyScript.tab/ОВиК.panel/MarksOn3D.pushbutton/README.md)

### 📏 MarksLength — Автодлина марок

**Расположение:** `BIM_Rage_4er.extension/pyScript.tab/ОВиК.panel/MarksLength.pushbutton`

**Версия:** 1.1

**Назначение:** Автоматическая корректировка длины полки марок на виде в зависимости от содержимого.

### 🔄 SortOV — Сортировка вентсистем

**Расположение:** `BIM_Rage_4er.extension/pyScript.tab/ОВиК.panel/SortOV.pushbutton`

**Версия:** 1.0

**Назначение:** Добавление префикса к параметру "Имя системы" систем воздуховодов по порядку от П к ДВ.

### 🧊 isolyshka — Проверка изоляции

**Расположение:** `BIM_Rage_4er.extension/pyScript.tab/ОВиК.panel/isolyshka.pushbutton`

**Назначение:** Вывод отчёта наличия изоляции систем воздуховодов на текущем виде.

### 🔒 Закрепление осей

**Расположение:** `BIM_Rage_4er.extension/pyScript.tab/Общая.panel/Закрепление осей.pushbutton`

**Назначение:** Автоматическое закрепление всех осей в проекте.

### ✏️ Именование видов

**Расположение:** `BIM_Rage_4er.extension/pyScript.tab/Общая.panel/Именование видов.pushbutton`

**Назначение:** Пакетное переименование видов с добавлением суффикса, префикса и номера.

## 🖥 Установка и использование

Чтобы начать пользоваться скриптами, следуйте простым инструкциям:

1. Установите расширение PyRevit для Revit.
2. Скачайте необходимые скрипты из данного репозитория.
3. Разместите скачанные файлы в нужной директории PyRevit.
4. Активируйте плагин и пользуйтесь новыми инструментами!

### Структура установки

```
%appdata%/pyRevit/
└── Extensions/
    └── BIM_Rage_4er.extension/
        └── pyScript.tab/
            └── ОВиК.panel/
                ├── MarksOn3D.pushbutton/
                ├── MarksLength.pushbutton/
                ├── SortOV.pushbutton/
                └── isolyshka.pushbutton/
```
