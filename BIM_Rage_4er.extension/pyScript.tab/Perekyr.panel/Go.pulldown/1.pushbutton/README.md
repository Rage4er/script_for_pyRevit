# 1_pushbutton - Анимированная кнопка с мячами и частицами

## Описание

Этот скрипт реализует кнопку с округлыми краями и анимацией мячей и частиц. При нажатии на кнопку запускается динамическая анимация с движущимися объектами, создавая визуальный эффект. Подходит для использования в интерфейсе pyRevit.

**Основные функции:**
- 🎨 Округлые кнопки с настраиваемыми цветами
- 🎮 Анимация мячей с физикой
- 🌌 Эффект частиц
- ⚙️ Настройка гравитации и trails
- 🔄 Возможность добавления новых мячей

## Функционал

### Кнопка с округлыми краями
- Реализована через класс `RoundedButton`
- Поддерживает изменение цвета при наведении мыши
- Отрисовка с плавными краями

### Анимация мячей
- Мячи движутся по правилам физики
- Поддерживают столкновения между собой
- Можно добавлять новые мячи вручную

### Эффект частиц
- Создаются при движении мячей
- Динамически обновляются
- Настраиваемый размер и цвет

## Классовые структуры

1. **RoundedButton**
   - Инициализация с радиусом скругления
   - Обработка событий мыши
   - Метод `OnPaint` для отрисовки

2. **AdvancedAnimationForm**
   - Основное окно анимации
   - Содержит настройки гравитации и trails
   - Управляет таймером анимации

3. **Particle**
   - Базовый класс для частиц
   - Обновление позиции и отрисовка

4. **Ball**
   - Мячи с физикой
   - Поддержка столкновений
   - Методы для движения и отрисовки

## Настройка

- Гравитация: можно изменить через панель настроек
- Trails: включить/выключить след мячей
- Добавление мячей: через кнопку "Добавить мяч"

## Как использовать

1. Поместите скрипт в панель `1.pushbutton` в pyRevit
2. Запустите скрипт через интерфейс pyRevit
3. Настройте параметры гравитации и trails
4. Нажмите кнопку для запуска анимации
5. Добавьте новые мячи при необходимости

## Технические детали

- **Фреймворк**: Python (IronPython) с pyRevit
- **UI**: Windows Forms для анимационного окна
- **Зависимости**: 
  - pyRevit для интеграции
  - System.Windows.Forms для GUI элементов
