# AutoView - Автоматическое размещение 3D видов на листы (версия 6.0)

## Описание

Скрипт `AutoView_script.py` предназначен для автоматического размещения 3D видов на листы Revit с оптимальным заполнением пространства. Скрипт группирует виды по настраиваемым системам и использует алгоритм упаковки прямоугольников MaxRects с подбором оптимальных комбинаций для каждого листа.

**Ключевые возможности версии 6.0:**
- 🎨 **Визуализация процесса** — анимация поиска (лупа/фонарик) и прогресс-панель с сеткой видов
- 📊 **Окно результатов** с итоговой статистикой
- 🔧 **Гибкая настройка групп** — 5 настраиваемых групп с выпадающими списками и пресетами
- 🚀 **Трехпроходной алгоритм** обнаружения границ (100px → 25px → 5px)
- ⚡ **Уменьшенное разрешение** экспорта (1000×1000 px)
- ⏱ **Замеры времени** на всех этапах
- 🪶 **Уменьшенные калибровочные квадраты** (5×5 мм)

## Процесс работы (по шагам)

### Шаг 1: Выбор листа-шаблона
Пользователь выбирает лист в Revit и запускает скрипт.

### Шаг 2: 🔍 Анимация "Чтение листа-шаблона"
Тёмное окно с вращающейся лупой и лучом прожектора. Считывается BoundingBox, формат, параметры штампа (~1-2 сек).

### Шаг 3: ⚙️ Настройка групп
Форма с 5 настраиваемыми группами. Каждая группа — до 10 префиксов через выпадающие списки. Доступны пресеты: П+В, ДП+ДВ, А+У / П, В, ДП+ДВ, А+У / П, В, ДП, ДВ, А, У.

### Шаг 4: 🔍 Анимация "Сбор видов из проекта"
Сбор 3D видов, извлечение префиксов, группировка, сортировка (~3-5 сек).

### Шаг 5: ☑️ Форма выбора видов
Виды отображаются сгруппированными по системам. Настройка отступов, зазоров, высоты штампа.

### Шаг 6: 🎯 Прогресс-панель "Анализ и размещение"
Тёмная панель с сеткой видов (🟢 готов, 🟠 в процессе, ⚫ ожидает), прогресс-баром и процентами. Для каждого вида: создание калибровочного листа → экспорт PNG → улучшение изображения → поиск границ → удаление листа.

### Шаг 7: 📦 Размещение на листах
Оптимальная упаковка с подбором комбинаций, создание листов, размещение Viewport'ов с учётом штампа.

### Шаг 8: ✅ Окно результатов
Итоговая статистика: создано листов, размещено видов, пропущено видов.

## Алгоритм обнаружения границ

Трехпроходной поиск с адаптивным порогом:
1. **Оценка фона** — медиана яркости по краям изображения
2. **Проход 1** — шаг 100px по всей области
3. **Проход 2** — шаг 25px в расширенной области (fallback: по всей)
4. **Проход 3** — шаг 5px по периметру для точного уточнения
5. **Адаптивный порог**: тёмный фон → `b > bg+20`, светлый → `b < bg-20`

## Настройки

| Параметр | Значение | Описание |
|----------|----------|----------|
| EXPORT_PIXELS | 1000 | Разрешение экспорта |
| PADDING_MM | 5 | Отступ вокруг содержимого |
| CALIBRATION_SQUARE_MM | 5.0 | Размер калибровочного квадрата |
| MM_PER_PX | 1.5 | Коэффициент мм/пиксель |
| STAMP_WIDTH_MM | 100 | Ширина штампа |
| STAMP_HEIGHT_MM | 50 | Высота штампа |
