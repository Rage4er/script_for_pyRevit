# Mari0 Platformer Game - Игровой платформер на Python для pyRevit

## Описание

Простой 2D платформер на Python для pyRevit. Игрок управляет персонажем (в стиле Марио), собирает монеты, избегает врагов и достигает флага. Игра включает базовую физику, систему столкновений и счетчик очков.

**Ключевые возможности:**
- 🎮 Классические платформерные механики (движение влево/вправо, прыжки)
- 🪙 Система сбора монет с подсчетом очков
- 👾 Искусственный интеллект врагов с базовыми паттернами движения
- 🏁 Флаг как цель завершения уровня
- 🎨 Процедурная генерация уровня
- 📊 Отображение счета и статуса игры

## Механика игры

### Управление
- **Стрелки влево/вправо**: Движение персонажа
- **Пробел**: Прыжок

### Основные системы
1. **Движение персонажа**
   - Физика с гравитацией
   - Управление скоростью
   - Прыжки с проверкой на контакт с землей

2. **Обнаружение столкновений**
   - Обработка столкновений с платформами
   - Сбор монет
   - Встречи с врагами (уничтожают игрока или дают бонус прыжка)

3. **Генерация уровня**
   - Случайное создание платформ
   - Случайное расположение монет и врагов
   - Позиционирование флага в конце уровня

## Прогресс игры

1. **Стартовое окно**: Отображает название игры и инструкции
2. **Генерация уровня**: Создает платформы, монеты и врагов
3. **Основной игровой цикл**:
   - Обновление движения и физики
   - Проверка столкновений
   - Обновление счета
4. **Конец игры**: Активируется при столкновении с врагом
5. **Победа**: Достижение флага

## Технические детали

- **Фреймворк**: Python (IronPython) с интеграцией в pyRevit
- **UI**: Windows Forms для игрового окна
- **Зависимости**:
  - pyRevit (если используется интеграция с Revit)
  - System.Windows.Forms для элементов интерфейса
- **Требования**:
  - Revit 2021+ (если используется интеграция с pyRevit)
  - Python-среда с необходимыми библиотеками

## Как запустить

1. Убедитесь, что pyRevit установлен и настроен
2. Поместите скрипт в панель pyRevit (например, в `Mari0.pushbutton`)
3. Запустите скрипт через интерфейс pyRevit
4. Используйте стрелки и пробел для игры
