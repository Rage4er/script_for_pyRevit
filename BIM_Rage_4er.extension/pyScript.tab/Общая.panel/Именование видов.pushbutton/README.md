# BIM_Rage_4er - Расширение для pyRevit

## Описание

BIM_Rage_4er - это расширение для [pyRevit], платформы для автоматизации в Autodesk Revit. Расширение предоставляет набор скриптов для упрощения работы с видами, осями и другими элементами в Revit-проектах.

## Установка

1. Скачайте или клонируйте репозиторий с расширением.
2. Разместите папку `BIM_Rage_4er.extension` в директории расширений pyRevit (обычно `%APPDATA%\pyRevit\Extensions`).
3. Перезапустите Revit или перезагрузите pyRevit для применения изменений.

## Структура расширения

- `pyScript.tab` - Основная вкладка в интерфейсе pyRevit.
  - `Общая.panel` - Панель с общими инструментами.
    - `Закрепление осей.pushbutton` - Инструмент для закрепления осей.
    - `Именование видов.pushbutton` - Инструмент для пакетного переименования видов (подробная инструкция ниже).
  - `Perekyr.panel` - Панель с инструментами для Perekyrа.
  - `ОВиК.panel` - Панель для ОВиК.

## Инструкция к скрипту "Именование видов"

### Описание
Скрипт "Именование видов" позволяет выполнять пакетное переименование выбранных видов в активном документе Revit. Это полезно для стандартизации именования видов в больших проектах.

### Как использовать
1. Откройте документ Revit.
2. Выделите виды в Навигаторе проекта или в виде.
3. Нажмите на кнопку "Именование видов" в панели "Общая" вкладки pyScript.
4. В открывшемся окне:
   - **Выбор видов**: Используйте "Обновить выделение" для загрузки предварительно выбранных видов или "Добавить к выделению" для ручного добавления видов.
   - **Настройки именования**:
     - **Префикс**: Текст, добавляемый в начало имени.
     - **Суффикс**: Текст, добавляемый в конец имени.
     - **Начальный №**: Номер, с которого начнется нумерация (по умолчанию 0).
     - **Формат номера**: Выберите формат (1, 01, 001) для ведущих нулей.
     - **Заменить текущее имя**: Если отмечено, текущее имя будет полностью заменено префиксом, номером и суффиксом. Если нет, номер добавится к текущему имени.
   - **Предпросмотр**: Нажмите "Обновить предпросмотр" для просмотра предполагаемых изменений.
   - **Выполнение**: Нажмите "ВЫПОЛНИТЬ ПЕРЕИМЕНОВАНИЕ" для применения изменений.
5. Закройте окно после завершения.

### Примечания
- Все изменения выполняются в транзакции Revit, что позволяет отменить операцию при необходимости.
- Логи операций выводятся в поле результатов.
- Инструмент поддерживает только виды (Views), другие элементы игнорируются.
