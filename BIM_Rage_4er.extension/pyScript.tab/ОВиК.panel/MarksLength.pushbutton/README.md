# MarksLength - Автоматическая корректировка длины марок

## Описание

Скрипт `MarksLength_script.py` предназначен для автоматической корректировки длины полки (выноски) марок на выбранных видах в Autodesk Revit в зависимости от содержимого марки. Это помогает оптимизировать отображение марок, делая их более читаемыми и компактными.

## Инструкция

1. Откройте активный документ Revit.
2. Запустите скрипт, нажав на кнопку "MarksLength" в панели "ОВиК" вкладки pyScript.
3. В появившемся окне мастера следуйте вкладкам:
   - **Вкладка 1**: Выберите виды для обработки (3D-виды или другие). Используйте поиск и выбор/снятие всего.
   - **Вкладка 2**: Выберите категории элементов для обработки.
   - **Вкладка 3**: Выберите параметры, которые будут влиять на длину полки (например, параметры марки).
   - **Вкладка 4**: Настройте опции генерации (если применимо, основываясь на файле `tag_defaults.json`).
   - **Вкладка 5**: Просмотрите сводку и нажмите "Выполнить".
4. Скрипт автоматически скорректирует длину полки марок на основе содержимого (например, длины текста или количества символов).
5. Просмотрите изменения на видах. При необходимости отмените транзакцию через Undo в Revit.

## Примечания

- Скрипт использует файл `tag_defaults.json` для сохранения настроек по умолчанию.
- Работает в транзакции Revit для обеспечения возможности отмены.
- Поддерживает только определенные типы марок и видов (проверьте поддерживаемые категории в интерфейсе).
- Если параметры не найдены, скрипт может пропустить некоторые элементы.
