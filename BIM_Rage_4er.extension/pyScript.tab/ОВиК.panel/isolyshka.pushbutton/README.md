# isolyshka - Отчет по изоляции воздуховодов

## Описание

Скрипт `isolyshka_script.py` предназначен для вывода отчета о наличии изоляции в системах воздуховодов в активном виде Autodesk Revit. Он собирает элементы категории "Воздуховоды (сегменты)" и "Соединительные детали воздуховодов", извлекает информацию о типе изоляции, категориях элементов и группирует их по имени системы. Отчет выводится в виде таблицы в окне вывода pyRevit.

## Инструкция

1. Откройте активный документ Revit и перейдите на нужный вид (план, разрез и т.д.).
2. Запустите скрипт, нажав на кнопку "isolyshka" в панели "ОВиК" вкладки pyScript.
3. В появившемся диалоговом окне подтвердите запуск проверки, выбрав "Yes" (Да).
4. Скрипт автоматически обработает элементы активного вида и выведет отчет:
   - Группы по имени системы.
   - Таблицы с идентификатором элемента (ссылка на элемент), категорией и типом изоляции.
5. Просмотрите результаты в окне вывода pyRevit. Элементы без изоляции отмечены как "Нет изоляции ⛔".

## Примечания

- Скрипт работает только с элементами активного вида.
- Использует параметры "Имя системы" и "Тип изоляции".
- Для каждого элемента без изоляции выводится соответствующее сообщение.
- Поддерживаемые категории: "Воздуховоды (сегменты)" и "Соединительные детали воздуховодов".
